# Mentenaz Forge

Welkom by Mentenaz Forge, 'n Next.js-projek wat met `create-next-app` gebootstrap is.

## Begin

Eerstens, hardloop die ontwikkelingsbediener:

```bash
npm run dev
# of
yarn dev
# of
pnpm dev
# of
bun dev
```

## Projek Struktuur

Die projek bestaan uit die volgende mappen:

* `src`: Die bronkode vir die projek.
* `node_modules`: Die afhanklikhede vir die projek.
* `public`: Die publieke bestande vir die projek.

## Konvensies

* Gebruik camelCase vir variabele- en funksienaam.
* Gebruik PascalCase vir klasname.
* Gebruik snake_case vir bestandsname.
